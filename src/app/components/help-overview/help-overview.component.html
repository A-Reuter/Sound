Info

Sound is a tool for simulating and visualizing firing sequences within workflow nets to support users in verifying the nets conformance to the soundness property as defined by Wil M. P. van der Aalst.


It is of utmost importance to note that this tool will never verify that a given petri net is sound. Instead, it will only ever verify that a given net is not sound, specifically when a net marking is reached which does not conform to the criteria of the soundness property.


Main Features
  • interactive net visualization
  • testing workflow net requirements
  • manual firing of individual transitions
  • automated execution of random firing sequences
  • replaying entries from a log of completed firing sequences
  • checking the conformance of a marking to the soundness property

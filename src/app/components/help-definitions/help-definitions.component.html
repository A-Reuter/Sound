Definitions

this section lists the core definitions the checks implemented in this application are based on

Petri Net

a petri net PN = (P, T, F) is a triple consisting of:

  • P - a finite set of place nodes
  •  place nodes have a marking consisting of individual tokens
  •  the set of all place markings defines the state of the net
  •  (in a petri net modelling a process, places represent partial states)
  • T - a finite set of transition nodes
  •  transition nodes have a set of input places, which are connected to the transition by an arc leading from the place to the transition
  •  transition nodes have a set of output places, which are connected to the transition by an arc leading from the transition to the place
  •  transition nodes have a state: if every input place of the transition has a marking equal to or greater than the weight of the arc leading to the transition, the transition is enabled
  •  transition nodes are fireable: an enabled transition can be fired to consume a number of tokens from every input place and produce a number of tokens on every output place, each equal to the respective arc weight
  •  (in a petri net modelling a process, transitions represent activities)
  • F - a set of arcs representing the flow relation
  •  arcs can either lead from a place to a transition or from a transition to a place, they can never connect two nodes of the same type
  •  arcs have a weight that dictates marking changes when firing the transition connected to an arc
  •  (in a petri net modelling a process, the set of arcs connected to a singular transition node represent the transitional step from the partial state represented by the input places of the transition to the partial state represented by the output places of the transition that is triggered by executing the activity represented by the transition)

Workflow Net

a petri net is a workflow net if and only if:

  1.  (i)  it has exactly two special places:
    1. (1) a source place i, which has no incoming arcs
    2. (2) a sink place o, which has no outgoing arcs
  3. (ii)  for each transition t and place p there exists at least one directed path from i to o that includes t / p

Soundness

let I be the start marking and O be the end marking of a workflow net
for I, let only the source place i be marked by a singular token
for O, let only the sink place o be marked by a singular token

a workflow net is sound if and only if:

  1.   (i)  for every state M reachable from state I, there exists a firing sequence leading from state M to state O
  2.  (ii)  there is only one state reachable from state I that contains a token on place o, which is state O
  3. (iii)  for each transition t there exists at least one state reachable from state I that enables t
